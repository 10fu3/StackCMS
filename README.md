# README

Headless CMS software

This CMS can...
- define your original contents field
- relation your contents to contents
- search contents by filter (called mongo db api)

## based on
- golang
  - gin (web framework)
- maria db (mysql)
  - contents field management
  - account management
  - permission management
  - role management
- mongo db
  - contents maneged
